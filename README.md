# Codigo Arduino para IMU MPU9250 
Código utilizado para captar os dados fornecidos pela IMU MPU 9250 e realizar o envio através de um dispositivo NODEMcu para um banco de dados do tipo Firebase


# Requirements
* FirebaseArduino.h
* SPI.h
* ESP8266.h
* Wire.h
* Arduino UNO.
* NodeMCU
* IMU MPU 9250
